# Code of Conduct

This project adopts the **[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
(version 2.1) as its code of conduct. The full text is available at that link.

## In short

Be respectful and constructive. We want participation in this project — issues, pull
requests, discussions, and reviews — to be a welcoming experience for everyone, regardless of
background or experience level. Assume good faith, keep feedback focused on the work, and help
others contribute.

## Reporting

If you experience or witness unacceptable behavior, please report it privately to the project
maintainer via their GitHub profile [@uchimata2](https://github.com/uchimata2). Reports will be
handled discreetly. (Maintainers: replace this with a dedicated contact address if one becomes
available.)

All maintainers are expected to follow and enforce this code of conduct in good faith.
